# Lesson10 - 碰撞检测与简易物理系统（含碰撞响应）

## 学习目标

1. 手写实现 AABB、圆碰撞检测；
2. 实现碰撞响应（位置修正、速度变化）；
3. 区分物理阻挡与触发区。

## 重点理解

- 碰撞检测 vs 响应：检测只告诉你“是否碰撞”，响应回答“如何分开并改变运动”。
- AABB 分离：计算重叠深度，沿最小位移方向修正位置。
- 简单响应：对静止物体（地面），将玩家位置移到接触边缘，并将垂直速度置零。
- 触发区：不产生物理阻挡，只产生逻辑事件（如拾取金币）。

## 动手练习题

1. 封装 AABB、点-矩形、圆-圆碰撞检测函数。
2. 实现平台跳跃物理：
   - 重力（每帧 `vy \+= gravity \* delta`）。
   - 碰撞地面后 `is\_grounded = True`，并将玩家 y 坐标修正到地面之上。
   - 按空格跳跃时 `vy = \-jump\_power`。
3. 区分“玩家 vs 方块”阻挡与“玩家 vs 金币”触发。
