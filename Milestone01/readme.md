# 综合案例 01 —— 极简桌面交互画板（带时间特效）

## 学习目标

1. 综合运用主循环 + delta time + 渲染 + 输入；
2. 实现画笔绘制、图形、颜色切换，以及时间驱动特效（残影淡出、背景缓慢变色）；
3. 验证时间系统对特效帧率解耦的效果。

## 重点理解

- 主循环统一调度：输入采集 → 逻辑更新（delta） → 渲染。
- 生命周期管理：每个绘制的图形存储出生时间，每帧用 delta 累加存活时间，超过寿命则删除。

## 动手练习题

实现以下功能：

1. 窗口循环、FPS 显示、正常退出。
2. 鼠标左键拖拽绘制连续线条（存储点阵）；按 R 键切换矩形/圆形笔刷。
3. 时间驱动特效：
   - 每个绘制的图形/点 2 秒后自动淡出并移除。
   - 背景色随实际运行时间缓慢循环（HSV 色相每 10 秒转一圈）。
4. 交互：数字键 1/2/3 切换颜色，C 清空，Ctrl+Z 撤销。
5. 扩展：实现“残影拖尾”——鼠标移动轨迹上保留半透明痕迹。

## 🔁 映射引擎原理

- 思考 Godot 中 `Tween` 如何实现一个对象的淡出。你手写的“生命周期+alpha 递减”本质是简化的 Tween。
