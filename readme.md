**理解游戏引擎**

“拆引擎、学底层、补内功”

本项目使用Pygame库来理解游戏引擎基本原理，让你知道「现代引擎背后为什么这么设计」。

分为如下课程:

- Lesson 1:  游戏主循环
- Lesson 2:  基础渲染与图形管线
- Lesson 3:  输入系统与交互判定
- Lesson 4:  有限状态机 FSM
- Lesson 5:  资源管理与数据驱动
- Lesson 6:  碰撞检测与物理基础
- Lesson 7:  动画、帧动画、时间系统
- Lesson 8:  架构分层与模块解耦

注：

* 每一个课程，只做一个 1～300 行的小 Demo，做完就归档，不迭代成大项目。
* 不用任何 UI 框架、物理库、动画库，全原生实现，目的是「理解底层」
* 推荐学完立刻反向映射到实际项目/引擎中，帮助理解引擎「为什么这么设计」
