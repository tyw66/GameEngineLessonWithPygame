# Lesson11 - 动画系统与时间驱动逻辑

## 学习目标

1. 理解动画本质：时间驱动的属性变化；
2. 实现帧动画（精灵表切割）、缓动动画、全局计时器；
3. 溯源 Godot `AnimationPlayer` 与 `Tween`。

## 重点理解

- 帧动画：将精灵表分成多帧，每帧停留固定时间（用 delta 累加切换）。
- 缓动动画：数学插值函数（线性、缓入缓出）计算属性中间值。
- Delta 时间：确保动画速度与帧率无关。

## 动手练习题

1. 实现精灵行走动画：根据方向和时间切换帧，支持播放、暂停、循环。
2. 手写缓动函数库（至少线性、Quad 缓入缓出），实现按钮 hover 缩放、方块的平滑移动动画。
3. 实现计时器：创建计时器类，基于 delta 倒计时，倒计时结束触发回调。
