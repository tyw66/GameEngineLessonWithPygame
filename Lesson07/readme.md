# Lesson07 - 有限状态机 FSM（逻辑架构基石）

## 学习目标

1. 掌握状态完整生命周期（enter/update/exit）；
2. 实现 3 种复杂度 FSM：if-else、GoF 状态模式、可中断状态；
3. 理解状态解耦与全局中断逻辑。

## 重点理解

- 状态生命周期：进入时初始化 → 每帧更新 → 退出时清理。
- GoF 状态模式：每个状态独立成类，状态机维护当前状态。
- 全局中断：如受伤、死亡等应能强制切换当前状态，通常用一个「高优先级状态」或状态栈实现。

## 动手练习题

1. 枚举 + if-else 实现角色「站立/行走/跳跃」三状态。
2. 面向对象实现标准状态模式，每个状态独立成类。
3. 进阶：增加「受伤」中断状态，结束后恢复原状态。
